How to create a certificate

edtFTPj/PRO does not provide any tools for creating certificates, however several good tools are available on the Internet for free.  Most popular is OpenSSL.  Please refer to the following links for instructions on how to use OpenSSL to create keys and certificates:
·    Keys: http://www.openssl.org/docs/HOWTO/keys.txt
·    Certificates: http://www.openssl.org/docs/HOWTO/certificates.txt

The topics, Public Key Cryptography and Certificates and Certificate Authorities (CAs), present introductions to public keys and certificates.

When creating client certificates note that edtFTPj/PRO supports certificates in PEM and Java keystore formats.
