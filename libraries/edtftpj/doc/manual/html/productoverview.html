Product Overview



edtFTPj/Free offers a set of robust and powerful, yet easy-to-use classes that make transferring files using the FTP protocol a breeze. 

edtFTPj/Free is the world's most popular Java FTP library. It is widely tested and has an active user community.

edtFTPj/Free's features include:

·     Rapid and Robust File Transfers via the FTP protocol - one of the fastest Java FTP clients available.
·     Active and Passive modes, Binary and ASCII transfers.
·     Transfer Resumption for interrupted file transfers.
·     FTP Streams for flexible reading from and writing to FTP servers.

A more complete list of features is presented here.

edtFTPj/PRO, a commercial derivative, has additional features that include:

·     Secure File Transfers via the SFTP (FTP over SSH) and FTPS (FTP over SSL) protocols.
·     Directory transfers in one operation.
