bezier()

Draws a Bezier curve on the screen.


Source Code:

size(100, 100, P3D);
noFill();
stroke(255, 102, 0);
line(30, 20, 80, 5);
line(80, 75, 30, 75);
stroke(0, 0, 0);
bezier(30, 20, 0,  80, 5, 0,  80, 75, 0,  30, 75, 0);

size(100, 100, P3D);
noFill();
stroke(255, 102, 0);
line(85, 20, 10, 10);
line(90, 90, 15, 80);
stroke(0, 0, 0);
bezier(85, 20, 0, 10, 10, 0, 90, 90, 0, 15, 80, 0);
