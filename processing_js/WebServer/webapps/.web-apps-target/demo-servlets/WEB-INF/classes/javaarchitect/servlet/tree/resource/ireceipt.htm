What's iReceipt?

iReceipt is a portable electronic version of sales receipt obtained at points of sales or electronic ordering systems. iReceipt is a proof of sale which digitally signed with a possibility of digitally signed annotations. iReceipt is carrying all attributes of a regular receipt and accepted by any issuer as a regular one.

Why iReceipt?

Besides of being a proof of purchase,  iReceipt gives many benefits as for a merchant as for a consumer, like:

  1. Faster checkout process
  2. Faster,  easier and confident  return
  3. Faster and easier security check at stores where required
  4. Faster and easier rebate processing
  5. Faster and easier get recalls or compensations in case of class-action lawsuits
  6. Faster and easier manage warranties and claims
  7. Easier and more effective personal budget management
  8. Less untargeted junk mail

How iReceipt?

iReceipt can be as server as client based. Both versions have own benefits and can be combined.

Server iReceipt

A cash register at point of sale transmits iReceipt to a server using one of iReceipt supported secure protocols. iReceipt gets ownership based on an iReceipt id provided by a customer as in an electronic form as in a simple typing on a pad device. A customer can ask for a paper version of iReceipt and/or get it recorded on a client version of iReceipt.

Client iReceipt

It's an USB drive with iReceipt organized file system. It can be password protected. A customer can connect a client version of iReceipt to a special terminal of a cash register and get a copy of iReceipt. A customer can use a pin pad device for an optional entering a password.

Use iReceipt

All iReceipts get acknowledge of iReceipt issuer and can be used the same as a paper based receipts. Every time when a paper version of a receipt required, iReceipt can be used as well. A customer has a convenient access to own iReceipts using a web browser or a dedicated software.
