ChatHouse  V1.0
This chat servlet provides (100% HTML) based chat. The feature set includes, but not limited to:

  1. Multiroom chat capabilities with possibilities, as using predefined rooms in specified charset, as a user created rooms in default charset.
  2. Easy navigation between rooms with displaying room visitors.
  3. Locking rooms for new visitors and unlocking them when all left with chat log cleaning.
  4. Automatically dispose a user created room when number of visitors reached 0.
  5. Easy to provide SSL awared chat.
  6. Using push server capabilities offered by Microsoft and Netscape browsers with manual and automatic refresh features. Netscape 6 browser is not supported, because AOL/Netscape dropped this capability in the new browser. Shame on Netscape first proposed the server push model.
  7. A fully customizable L&F based on using template HTML pages.
  8. An ability to upload a file to a chat message with the following possibility to download the uploaded file by chat participants (GonnaSee® technology).
  9. An ability to send a message only to the selected chat participant.
  10. E-mail notification about chat rooms activities.
  11. Message board capabilities that allows an administrator of the chat publishing messages.
  12. All chat parameters are configurable without any code change.
  13. All in one instant installation, when you can start immediately after download without touching anything (Download&Play® technology). Preinstalled JDK 1.3 or above is required. Note that instant installing isn't provided anymore, instead of it web app package is provided.
